we need a GitHub token!

this extension talks to the GitHub API for resolving relative paths and although it works a little without a token, GitHub starts limitting it very soon and this extension stops being useful for that purpose.

you can get a personal GitHub API token at https://github.com/settings/tokens (don't need to authorize any scopes, a blank token is fine), then just paste it here:
